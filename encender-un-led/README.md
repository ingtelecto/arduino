# 💡 Encender un LED

Práctica básica para hacer parpadear un LED conectado a una placa Arduino cada segundo.

## 🧰 Materiales utilizados

- **1x** Placa Arduino (UNO, Nano, Mega o compatible)
- **1x** LED de 5mm (rojo, verde, azul o amarillo)
- **1x** Resistencia de 220Ω (para limitar corriente)
- **1x** Protoboard (breadboard)
- **2x** Cables Jumper (macho-macho)

## 📐 Circuito

1. Conecta el **ánodo** (pata larga del LED) al **Pin 8** de Arduino a través de la resistencia de 220Ω.
2. Conecta el **cátodo** (pata corta del LED) al pin **GND** de Arduino.

## 💻 Código

El código listo para cargar en Arduino IDE se encuentra en el archivo [`encender-un-led.ino`](encender-un-led.ino).
