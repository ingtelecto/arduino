# Guía de contribución

Gracias por colaborar con los proyectos de Arduino de Ingtelecto.

## Antes de comenzar

1. Crea un fork del repositorio.
2. Trabaja en una rama independiente.
3. Prueba el código en Arduino IDE.
4. Describe la placa y los componentes utilizados.
5. Envía un pull request explicando el cambio.

## Convención para las prácticas

Cada práctica debe incluir:

- Una carpeta con un nombre descriptivo.
- El archivo principal `.ino`.
- Una breve explicación del circuito.
- La lista de componentes utilizados.
- Comentarios dentro del código.
